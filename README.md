# Project Planner

Project Planner — это мобильное приложение для планирования и управления проектами, разработанное в рамках выпускной квалификационной работы.

## Основные возможности

- Создание и редактирование проектов

- Добавление задач и подзадач

- Назначение исполнителей

- Отслеживание статуса и сроков выполнения

- Авторизация и регистрация пользователей

- Просмотр профиля, друзей и списка проектов

- Уведомления о назначении задач

## Архитектура

Приложение реализовано по клиент–серверной архитектуре:

- Клиентская часть (mobile_app/) — Android-приложение на Kotlin с использованием Jetpack Compose.

- Серверная часть (server/) — REST API на Node.js с использованием фреймворка Express.js.

- База данных (database/) — PostgreSQL, обеспечивающая хранение данных о пользователях, проектах и задачах.

## Используемые технологии
- Frontend (мобильное приложение)	Kotlin, Jetpack Compose
- Backend (сервер)	Node.js, Express.js, bcrypt, pg
- База данных	PostgreSQL, pgAdmin
- Среды разработки	Android Studio, Visual Studio Code
- Контроль версий	Git, GitHub
